## ✨ 功能特色

### 🧠 智能推荐引擎
- **位次驱动排序** — 以考生位次为锚点（第 30 位），向上冲刺、向下保底，科学分配 96 个志愿
- **冲/稳/保梯度** — 自动计算录取概率，冲 30 / 稳 45 / 保 21（默认），比例随家庭策略动态调整
- **等效分换算** — 通过一分一段表精确换算跨年等效分数

### 🎯 张雪峰视角增强（v2.1）
- **家庭策略适配** — 富裕/普通/困难三档，影响冲稳保比例和推荐权重
- **天坑专业预警** — 生化环材等低回报专业自动标注 ⚠️
- **赛道推荐指数** — ⭐ 五星推荐，综合城市/赛道/办学性质打分
- **策略加权** — 考公方向（法学+25）、考研方向（基础学科+20）、就业方向（计算机+15）

### 🏥 体检与选科筛选
- 色觉检查（色盲/色弱级联过滤）
- 裸眼视力限制（飞行 ≥ 5.0、航海 ≥ 4.8）
- 公安类身高/体重硬性要求
- 选科要求精确匹配（3+1+2 模式）

### 📊 数据可视化
- 六爻图 — 城市分布/办学性质/录取概率多维度统计
- 城市级别分布、冲稳保占比一览

### 📤 多格式导出
- **Excel** (.xlsx) — 完整志愿表 + 排序标注
- **PDF** — 排版精良的打印版
- **Word** (.docx) — 可二次编辑

## 🚀 快速开始

### 环境要求

- Python 3.10+
- Windows 10/11
- 4GB+ 内存

### 方式一：一键安装（推荐）

1. 双击 `setup.bat` — 自动创建虚拟环境并安装依赖
2. 双击 `启动系统.bat` — 启动程序

### 方式二：手动安装

```bash
git clone https://github.com/lpzhawei/gaokao-volunteer.git
cd gaokao_volunteer

python -m venv venv
venv\Scripts\activate      # Windows
pip install -r requirements.txt
```

### 运行

```bash
python main.py
```

或双击 `启动系统.bat`

> 数据库已内置（`data/db/gaokao.db`），包含 2023-2025 年河北省投档数据，开箱即用。

### 打包为 exe

```bash
pip install pyinstaller
pyinstaller gaokao.spec
# 输出在 dist\河北高考志愿填报系统\
```

## 📋 使用流程

```
┌──────────────┐    ┌──────────────┐    ┌──────────────┐    ┌──────────────┐
│  1. 导入数据  │───▶│  2. 填写信息  │───▶│  3. 生成志愿  │───▶│  4. 导出结果  │
│  投档Excel    │    │  分数/位次    │    │  96志愿推荐   │    │  Excel/PDF   │
│  一分一段表   │    │  选科/体检    │    │  冲稳保梯度   │    │  Word        │
└──────────────┘    └──────────────┘    └──────────────┘    └──────────────┘
```

1. **导入数据** — 「数据管理」页 → 选择河北省投档 Excel 文件 → 设置年份/科目 → 导入
2. **填写信息** — 输入 2026 预估分、位次（必填）、选科、体检条件、家庭策略
3. **生成志愿** — 点击「开始生成 96 个志愿」→ 自动跳转结果页
4. **导出结果** — 选择 Excel / PDF / Word 格式导出

### 核心算法

```
考生位次 65,000（锚定第 30 位）
  │
  ├── 第 1~29 位  冲刺方向（专业位次 < 考生位次，越难越前）
  │     位次差 +19,417 → 位次差 +291
  │
  ├── 第 30 位    锚点（最接近考生位次）
  │     位次差 ≈ 0
  │
  └── 第 31~96 位 保底方向（专业位次 > 考生位次，越稳越后）
        位次差 -291 → 位次差 -76,409
```

| 指标 | 说明 |
|------|------|
| 排序轴 | 专业最低位次（min_rank），仅用 2025 年数据 |
| 录取概率 | Sigmoid 函数，基于位次差计算 |
| 梯度标签 | prob < 50% = 冲，50-80% = 稳，> 80% = 保 |
| 策略加权 | 张雪峰视角：城市/赛道/办学性质/天坑避坑 |
