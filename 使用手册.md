# 河北省高考志愿填报系统 使用手册

## 一、系统简介

本系统专为河北省2026年高考考生设计，基于近三年（2023-2025）历史录取数据，采用智能推荐算法为考生生成96个平行志愿方案。

**核心功能**：
- 根据考生分数/位次自动推荐志愿
- 冲稳保梯度科学分配（冲30/稳45/保21）
- 体检限制、选科要求自动过滤
- 支持省份、专业偏好筛选
- 导出Excel志愿表和PDF分析报告

## 三、操作步骤

### 第一步：录入考生信息

在【考生信息与推荐】页面填写：

| 字段 | 说明 |
|------|------|
| 姓名 | 用于导出文件命名 |
| 预估分数 | 高考成绩或预估分 |
| 政策加分 | 少数民族、独生子女等加分 |
| 首选科目 | 物理或历史（决定报考类别） |
| 再选科目 | 化学/生物/地理/政治（选2门） |
| 考生位次 | 可手动输入2026年位次，或留空系统自动查2025年位次 |
| 性别 | 用于体检限制判断 |
| 色觉 | 正常/色弱/色盲（影响医学、化工等专业） |
| 裸眼视力 | 用于公安、航海等专业判断 |
| 身高/体重 | 用于公安、飞行等专业判断 |

### 第二步：设置筛选条件（可选）

#### 省份偏好
- 可添加多个意向省份
- 勾选"放宽"后，若候选池不足会自动扩大到全国

#### 专业偏好
- 支持关键词匹配（如"计算机"、"医学"、"法学"）
- 勾选"放宽"后，若候选池不足会自动取消专业限制

#### 位次范围滑块
- **冲方向**：向位次更好（排名更高）的学校扩展多少名
- **保方向**：向位次更差（排名更低）的学校扩展多少名
- 默认自动计算，可手动调整

#### 其他选项
- 院校性质：公办/民办/独立学院

### 第三步：生成志愿

点击 **「生成志愿」** 按钮，系统自动：
1. 根据分数/位次查询考生排名
2. 按位次范围筛选候选专业
3. 应用省份、专业偏好过滤
4. 体检限制、选科要求自动排除
5. 按2025年位次排序，分配冲稳保梯度
6. 输出96个志愿

### 第四步：查看结果

在右侧表格查看志愿列表：
- 梯度颜色：红=冲、绿=稳、蓝=保
- 风险等级：高/中/低
- 筛选：可按梯度、风险、关键词筛选

### 第五步：导出文件

#### 导出Excel
- 包含完整96志愿 + 考生个人信息
- 表头：姓名、预估分、加分、位次、科目等
- 数据列：序号、梯度、院校、专业、省份、近三年分数、位次差、录取概率、风险、2025位次、选科要求、体检限制

#### 导出PDF报告
- 考生基本信息
- 志愿梯度分布统计
- 96个志愿明细表
- 免责声明

## 四、推荐算法说明

### 排序规则
1. 按**2025年最低位次**升序排列（位次越小越靠前）
2. 找到与考生位次最接近的专业作为锚点
3. 锚点放在第30位，向上取29条冲刺志愿，向下取66条稳保志愿

### 梯度分配
| 梯度 | 数量 | 定位 |
|------|------|------|
| 冲 | 30个 | 录取概率20-50%，位次优于考生 |
| 稳 | 45个 | 录取概率55-85%，位次接近考生 |
| 保 | 21个 | 录取概率85%+，位次低于考生 |

### 过滤规则
1. **选科要求**：专业要求的再选科目必须在考生已选科目中
2. **体检限制**：
   - 色盲：排除医学、化工、美术等受限专业
   - 色弱：排除医学、化工等受限专业
   - 视力/身高/体重：排除公安、航海、飞行等受限专业
3. **院校性质**：民办/独立学院根据勾选过滤

## 五、注意事项

### ⚠ 重要提示
1. **本系统仅供参考**，最终以河北省教育考试院官方信息为准
2. 历史数据不代表未来录取结果，请合理评估风险
3. 96个平行志愿无专业调剂，投档后不符合条件会被退档
4. 建议结合专业就业前景、个人兴趣综合选择

### 常见问题

**Q: 位次查不到怎么办？**
A: 确保分数+加分在2025年一分一段表范围内（140-693分），或手动输入2026年位次

**Q: 生成的志愿少于96个？**
A: 尝试勾选"放宽"省份/专业偏好，或调整位次范围滑块扩大范围

**Q: 导出的文件在哪里？**
A: 打开 `%LOCALAPPDATA%\河北高考志愿填报系统\output\` 文件夹

**Q: 如何查看运行日志？**
A: 打开 `%LOCALAPPDATA%\河北高考志愿填报系统\logs\app.log`
